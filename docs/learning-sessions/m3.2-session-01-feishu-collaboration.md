# M3.2 学习复盘 01：Feishu 审核协同（通知 + 回调 + 提醒）

## 1. 本次实现了什么
- 增加 Feishu 通知器：待审核通知、发布结果回执、审核提醒。
- 增加 Feishu 本地回调服务（2B）：接收审核动作并写入 `outputs/review-instructions/*`。
- 增强审核指令模型：支持 `action/source/operator/traceId/messageId` 审计字段。
- CLI 增加两类入口：`--serve-feishu-callback`、`--notify-review-reminder`。
- 提醒策略落地：周一 11:30（Asia/Shanghai）单次提醒，按 marker 去重。

## 2. 流程图（M3.2 第一阶段）
```text
weekly run
  -> persist review artifact
  -> notify pending review to Feishu

feishu callback (local server + tunnel)
  -> verify token/signature
  -> parse action payload
  -> append review instruction
  -> recheck/watchdog consumes latest instruction

weekly 11:30 reminder job
  -> scan pending review artifacts
  -> skip sent markers
  -> send Feishu reminder
```
